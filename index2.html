Project Introduction

The Smart Irrigation System is an innovative solution designed to automate the watering process in agriculture using soil moisture sensors and microcontrollers. It helps in conserving water and improving efficiency. GOAL: monitor soil moisture and automaticall irrigate when dry, while allowing farmer to see live data and control the pump form anywhere via Iot.

Abstract

This system automates irrigation by monitoring soil moisture levels and activating water pumps only when needed. It reduces human labor and water wastage while enhancing crop growth and productivity.

Implementation

The project uses sensors to measure soil moisture, a microcontroller to process the data, and a relay module to control a water pump. When the soil is dry, the system turns on the pump automatically.

Working Principle

The moisture sensor sends real-time data to the controller. If moisture is below a threshold, the controller powers the pump. Once the required level is reached, the pump is turned off. This closed-loop system ensures efficient water usage.
